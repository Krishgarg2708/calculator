# Smart Calculator Pro

A **modern GUI calculator** made in **Python** using **Tkinter**.  
This is my **first Python GUI project**, featuring a dark neon theme, animations, keyboard support, and special buttons like clear & backspace.

---

## Features

- Dark + Neon glowing theme  
- Hover effects on buttons  
- Keyboard support  
- Backspace & Clear buttons  
- Smooth animations and modern font  
- Handles all basic operations: `+ - * / .`
